OmniDesk Ticketing System

Welcome to OmniDesk, a comprehensive ticketing system for customer support and issue tracking.

API Endpoints

POST /api/auth/login - User Login POST /api/auth/register - User Registration GET /api/tickets - List Tickets POST /api/client/submit-ticket - Submit Client Ticket About Page

Frontend: The main application interface is built with React and should be served separately on port 3000 during development.

Backend: This Flask API server handles authentication, ticket management, and data persistence.
